# wsl-clipboard.nvim

simple neovim plugin that copies yanked text to clipboard

# Installation

Lazy.nvim

```lua
{
    'hajin-chung/wsl-clipboard.nvim',
    config = function()
      require("wsl-clipboard").setup()
    end
},
```
